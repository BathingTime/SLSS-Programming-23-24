In Python, data can be grouped in categories called Types.

|Name|Example|
|---|---|
|boolean/bool|`True`,`False`|
|integer/int|`1`,`-1`,`0`|
|float|`1.5`,`-1.5`,`1.0`|
|string|`'hello'`|
|list|`[1,2,3]`|

An example of using these types of data:

```Python
fave_food='tempura'#string

my_age=17.5#float
```
